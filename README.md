# LearnerProject
Bir kurs merkezi projesi olarak hazırladığımız bu projede bir ASP.NET MVC 5.0 kulladık. ORM aracı olarak EntityFramework Code First yaklaşımını kullandık.

## Kullanılan teknolojiler:
-Asp.Net
-Entity Framework DbFirst
-SQL
-Html
-Css
-Bootstrap
-Linq
